## installation

-   have node installed.
-   use npm or yarn.
-   install packages
-   create .config folder
-   create .config/.env file
-   add `DISCORD_TOKEN=YOUR_BOT_TOKEN` to the env file
-   run (yarn/npm run) dev to start the bot
